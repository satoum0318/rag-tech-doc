# 🚨 至急修正: LM Studio設定

## 現在の問題

TechScoutで「記載がありません」と表示されるのは、**LM Studioのコンテキスト長が短すぎる**ためです。

現在: **4096トークン** （約3000文字）  
必要: **8192トークン以上** （約6000文字）

---

## 📝 修正手順（1分で完了）

### ステップ1: LM Studioを開く

### ステップ2: Local Serverタブに移動
左側の緑色アイコンをクリック

### ステップ3: サーバーを停止
「Stop Server」ボタンをクリック

### ステップ4: Context Lengthを変更
画面右側の設定で：

```
Context Length: 8192
```

に変更してください。

### ステップ5: サーバーを再起動
「Start Server」ボタンをクリック

### ステップ6: TechScoutを再起動
```powershell
# 現在のアプリを停止してから
.\run.ps1
```

---

## ✅ 修正後の効果

修正前：
```
❌ 「この文書に記載がありません」
（実際にはPDFに記載がある）
```

修正後：
```
✅ PDFの内容を正しく参照して回答
「永久磁石同期モータの温度推定には...」
```
