## 📝 ドキュメント追加方法

1. このフォルダ（`documents/`）に以下の形式のファイルを配置：
   - `.txt` - テキストファイル（UTF-8推奨）
   - `.pdf` - PDFファイル
   - `.docx` - Word文書

2. アプリを再起動：
   ```bash
   .\run.ps1
   ```

3. 自動的にインデックス化され、検索可能になります

## ⚠️ 注意事項

### 著作権について
- 学術論文を使用する場合は、著作権・利用規約を確認してください
- 機密文書は適切に管理してください
- 公開リポジトリの場合、センシティブな情報を含むファイルは配置しないでください

### ファイルサイズ
- 大きなPDFファイルは処理に時間がかかります
- 推奨: 1ファイルあたり10MB以下
- 大量のドキュメント（100個以上）の場合は、システムリソースに注意

### エンコーディング
- テキストファイルはUTF-8エンコーディングで保存してください
- 文字化けを防ぐため、BOM付きUTF-8を推奨
